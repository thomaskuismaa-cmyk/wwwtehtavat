📊 Funktioharjoitus

1️⃣ kolmensumma(luku1, luku2, luku3)

Kuvaus: Funktio saa kolme lukuparametria ja palauttaa niiden summan.

// Esimerkki:
kolmensumma(5, 10, 15) → 30
kolmensumma(2, 4, 6) → 12

Omilla arvoilla:

2️⃣ max(luku1, luku2)

Kuvaus: Funktio saa kaksi lukuparametria ja palauttaa niistä suuremman.

// Esimerkki:
max(25, 42) → 42
max(100, 50) → 100

Omilla arvoilla:

3️⃣ taulusumma(taulukko)

Kuvaus: Funktio saa parametrina lukuja sisältävän taulukon ja palauttaa taulukon lukujen summan.

// Esimerkki:
taulusumma([1, 2, 3, 4, 5]) → 15
taulusumma([10, 20, 30]) → 60
